# Proyecto: Tower Defense en C++

## Descripción
Evaluación integradora de listas secuenciales, listas doblemente enlazadas y listas simplemente enlazadas circulares. 

## Integrantes y División del Trabajo
* **Desarrollador 1:** Lista Secuencial (Sistema de Torres)
* **Desarrollador 2:** Lista Doblemente Enlazada (Sistema de Enemigos)
* **Desarrollador 3:** Lista Circular (Sistema de Oleadas)
* **Desarrollador 4 (Líder):** Motor del Juego, Menú y Colisiones
